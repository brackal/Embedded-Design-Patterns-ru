# Шаблоны проектирования для Embedded Systems

## [Шаблон Callback (Callback Pattern) ](https://github.com/brackal/Embedded-Design-Patterns/tree/main/CallbackPattern)

Шаблон Callback является классическим примером шаблона Observer.
Пример реализации шаблона Observer можно найти по ссылке: https://github.com/brackal/Head-First-Design-Patterns-Cpp

Шаблон Callback используется во встроенном программном обеспечении для реализации асинхронного взаимодействия между элементами программного обеспечения.
Например, когда нажатие кнопки должно асинхронно передаваться приложению.
